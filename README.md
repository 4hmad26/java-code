# Java Banking System (Console Application)

This is a basic Java console-based banking system project created to practice
core Java concepts such as Object-Oriented Programming, classes, methods, and
user input handling.

## Features
- Deposit money
- Withdraw money
- Check account balance
- Menu-driven console interface

## Concepts Used
- Java Classes & Objects
- Methods
- Conditional statements
- Scanner for input handling
- Basic OOP principles

## How to Run
1. Clone the repository
2. Compile the Java file:
   javac intro.java
3. Run the program:
   java intro
